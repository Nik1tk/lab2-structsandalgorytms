#Лабораторная работа №2

##Браузерная история. Выполнил студент группы ИДБ-25-07 Морозов Никита

Реализована система хранения истории браузера
Каждая запись содержит:
1. URL
2. Дату и время захода
3. Флаг закладки
---
Реализовано:
1. Навигация "назад/вперёд"
2. Поиск по домену
3. Очистка истории
4. Сохранение истории в файл Base-64
5. Вывод истории в виде таблицы
---
###Структура данных

За основу был взят двусвязный список, т.к. при его использовании есть ссылки на предыдущую и следующую страницы (prev, next), поэтому:
1.Любая навигация (перезод вперёд/назад) имеет сложность О(1)
2.При переходе назад и последующем переходе на новую страницу вся будущая история браузера удаляется, как и в обычных браузерах
---
###Пример работы
Для проверки работы программы используются страницы:
-https://google.com
-https://github.com
-https://stackoverflow.com
-https://youtube.com

После выполнения программы выводится:
-вся история
-навигация назад/вперёд
-загрузка истории в Base-64 и выгрузка истории из него
-очистка истории
